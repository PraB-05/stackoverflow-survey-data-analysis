# StackOverflow Developer Survey 2020 Analysis

This project performs exploratory data analysis (EDA) on the StackOverflow Developer Survey 2020 dataset to understand trends in programming languages, developer demographics, and salary distribution.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data cleaning and preprocessing
- Programming language popularity analysis
- Developer salary distribution analysis
- Data visualization of technology trends

## Dataset
StackOverflow Developer Survey 2020
